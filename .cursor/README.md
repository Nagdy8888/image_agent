# .cursor

Cursor IDE project configuration and rules.

## Contents

| Item | Description |
|------|-------------|
| `rules/` | Cursor rules (curriculum-learning, documentation-structure, error-log, folder-readme, folder-structure, phase-setup-guide, progress-diagram). |
| `plans/` | Implementation plan placeholders; add `.plan.md` files for features. |

Rules are applied by Cursor when relevant to the current task. See [Cursor docs](https://docs.cursor.com/context/rules-for-ai) for how to use rules.
