# frontend

Next.js App Router frontend for the Image Tagging Agent dashboard and search page.

| Folder / File | Purpose |
|---------------|---------|
| `app/` | App Router pages (page.tsx, layout.tsx, globals.css, search/page.tsx) |
| `components/` | React components (upload, overlay, tag cards, navbar, etc.) |
| `lib/` | Types, constants, utilities |
| `Dockerfile` | Production build for docker-compose |

Run locally: `npm run dev` (http://localhost:3000).
